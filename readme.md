# Dlang File Resources Library

## Supported File Types
- [x] BMP 
- [x] PNG (Read only)
- [x] JPEG (Baseline DCT Read only)
- [x] DDS (Read only)

- [x] LZ4 (Read only)
- [x] BZip2 
- [x] BZip3 
- [x] HGT Height data (read only)

- [x] WAV (Read only)

- [x] GLTF (Read only)
- [x] OBJ (Read only)

- [x] JSON5

## Requirements
- Dlang https://dlang.org/
